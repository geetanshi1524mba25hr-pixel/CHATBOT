#  Geetanshi's AI Assistant

A simple and elegant AI chatbot interface built using HTML, CSS, and JavaScript. This project demonstrates how to connect a custom UI with the OpenAI API to create a responsive conversational assistant.

##  Features

* Clean and modern chatbot UI
* Real-time AI responses
* Enter key message sending
* Gradient responsive design
* User & bot chat bubbles
* Error handling for API connection

##  Preview

This project includes a split-screen layout:

* Left side: JavaScript code connecting to OpenAI API
* Right side: Chatbot UI labeled **Geetanshi's AI Assistant**

##  Technologies Used

* HTML
* CSS
* JavaScript
* OpenAI API

##  How It Works

1. User types a message in input field
2. Message sent to OpenAI API
3. AI processes request
4. Response displayed in chat window

## Use Cases

* Personal AI assistant
* Student chatbot projects
* Frontend + API integration demo
* UI/UX chatbot design practice
